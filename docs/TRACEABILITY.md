# Traceability Report

result, algorithm, and implementation statement in the manuscript *"A
Privacy-Focused, Post-Quantum-Oriented Digital Identity System Using Blockchain
and Zero-Knowledge Proofs"* (Access-2026-15409) to the exact file(s) in this
repository that implement, produce, or validate it.

Legend for "Kind": **impl** = implementation; **measure** = produces the
measured number; **test** = validates; **data** = committed result/fixture;
**doc** = documentation. Status labels `[M]`/`[S]`/`[A]`/`[F]` match the paper.
